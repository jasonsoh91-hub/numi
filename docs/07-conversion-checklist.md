# 07 — Conversion Checklist: NUMI Landing Page

## Main Conversion Goal

Visitors submit their details to receive:
- The Pattern Code guide.
- Their first Pattern Insight.
- A pathway to go deeper with NUMI.

Primary CTA:
**Get My Free Pattern Insight**

---

# 1. Strategic Checklist

## Audience Fit
- [ ] The page clearly speaks to self-discovery seekers.
- [ ] The copy prioritises Spiritual Sophia as the primary ICP.
- [ ] The page also gives enough relevance to Seeking Sarah and Burned Out Brenda.
- [ ] The page does not try to target everyone equally.
- [ ] The page makes the lead magnet feel valuable.

## Positioning
- [ ] NUMI is positioned as pattern intelligence, not fortune telling.
- [ ] The Pattern Code is positioned as the first layer.
- [ ] NUMI is positioned as the deeper AI-powered interpretation layer.
- [ ] The page explains why NUMI is different from generic horoscopes, static reports, and wellness apps.
- [ ] The page avoids overpromising or deterministic claims.

## Offer Clarity
- [ ] The visitor understands what they get for free.
- [ ] The visitor understands what a Core Number is.
- [ ] The visitor understands what happens after submitting the form.
- [ ] The CTA is clear and low-friction.
- [ ] The page connects the free guide to the deeper NUMI product.

---

# 2. Hero Checklist

## Above-the-Fold
- [ ] H1 clearly states the main promise.
- [ ] Subheadline explains the offer in plain language.
- [ ] CTA appears above the fold.
- [ ] Secondary CTA appears above the fold.
- [ ] Lead magnet visual appears above the fold on desktop.
- [ ] Mobile users see CTA quickly without excessive scrolling.

## Hero Copy
- [ ] Uses “think, love, earn, decide” clearly.
- [ ] Mentions Core Number or Pattern Code.
- [ ] Does not use vague mystical language alone.
- [ ] Includes “free guide” or “free insight.”
- [ ] Includes reassurance: no prediction, self-understanding.

---

# 3. CTA Checklist

## CTA Consistency
- [ ] Primary CTA text is consistent: “Get My Free Pattern Insight.”
- [ ] Secondary CTA text is consistent: “See How It Works.”
- [ ] CTA appears in hero, lead magnet section, pattern section, form section, and final CTA.
- [ ] CTA buttons are visually prominent.
- [ ] Button text communicates value, not generic action.

## CTA Placement
- [ ] Hero CTA.
- [ ] Lead magnet intro CTA.
- [ ] How it works CTA.
- [ ] Pattern cards CTA.
- [ ] Product bridge CTA.
- [ ] Form CTA.
- [ ] Final CTA.

---

# 4. Lead Magnet Checklist

## Lead Magnet Value
- [ ] The Pattern Code cover/mockup is visible.
- [ ] The page explains the guide clearly.
- [ ] The page explains how to calculate Core Number.
- [ ] The page previews the 9 Pattern Codes.
- [ ] The page creates curiosity without giving away too much.
- [ ] The page says the guide reveals one layer and NUMI reveals the full map.

## Lead Magnet Framing
- [ ] Not framed as a generic PDF.
- [ ] Not framed as fortune-telling.
- [ ] Framed as a personal mirror.
- [ ] Framed as a quick self-discovery experience.
- [ ] Framed as an entry point into NUMI.

---

# 5. Copy Checklist

## Clarity
- [ ] Each section has one clear purpose.
- [ ] Headlines are specific.
- [ ] Body copy is short.
- [ ] Copy avoids dense paragraphs.
- [ ] Copy does not sound generic.

## Emotional Resonance
- [ ] The page validates repeated patterns.
- [ ] The page makes users feel seen.
- [ ] The page creates curiosity about their number.
- [ ] The page suggests there is a deeper layer to unlock.
- [ ] The page balances mystical and practical language.

## Risk Control
- [ ] Avoids deterministic claims.
- [ ] Avoids “your future is guaranteed” claims.
- [ ] Avoids medical, financial, or professional advice claims.
- [ ] Includes disclaimer in footer.
- [ ] Frames NUMI as self-reflection and personal growth.
